# Commissioning Checklist

## Before First Data

- Confirm asset ID, sensor locations, scaling, units, and sampling rate.
- Check accelerometer mounting torque or adhesive cure.
- Verify current sensor direction and offset.
- Confirm temperature sensor placement and expected ambient offset.
- Record baseline operating modes, speed ranges, loads, and product recipes.

## Baseline Run

- Run unloaded warm-up.
- Run normal loaded operation across representative speed bands.
- Confirm no maintenance work is in progress during baseline capture.
- Fit baseline and archive the baseline JSON with date, operator, and asset state.

## Fault Evidence Run

- Seed one condition at a time where safe and practical.
- Record physical intervention notes: imbalance weight, loosened bracket, belt adjustment, blocked cooling, or simulated acquisition issue.
- Re-run analytics and record detection delay and nuisance alarms.

## Handover

- Review alarm rationalization with operations.
- Attach action matrix to maintenance work-order process.
- Define who owns baseline updates.
- Define retention policy for raw telemetry and feature windows.
